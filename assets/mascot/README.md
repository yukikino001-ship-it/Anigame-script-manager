# 看板娘素材放置说明

V28.7 支持 4 个状态：

- idle：未执行任务时待机，建议 3 张。
- work：任务执行中，建议 2 张。
- rest：暂停、停止、完成后休息，建议 3 张。
- error：异常提示，建议 2 张。

推荐命名：

```text
assets/mascot/idle/idle_1.png
assets/mascot/idle/idle_2.png
assets/mascot/idle/idle_3.png
assets/mascot/work/work_1.png
assets/mascot/work/work_2.png
assets/mascot/rest/rest_1.png
assets/mascot/rest/rest_2.png
assets/mascot/rest/rest_3.png
assets/mascot/error/error_1.png
assets/mascot/error/error_2.png
```

也兼容直接放在 `assets/mascot/` 下的 `idle_1.png`、`work_1.png`、`rest_1.png`、`error_1.png` 等旧式命名。
